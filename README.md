## Installation

### Via Spicetify Marketplace (recommended)
1. Open Spicetify Marketplace
2. Search for "Cyrene HSR"
3. Click Install

### Manual installation
1. Download this repository
2. Copy the `Hsr3rdAnniversary` folder into your Spicetify Themes directory
3. Run the following commands:

4. spicetify config current_theme Hsr3rdAnniversary
spicetify config inject_css 1 replace_colors 1
spicetify apply
